# FC colortest
## overview
ファミコン用の色の表示テストプログラムです。一定時間おきに色が切り替わります。

## How to build

開発環境はCC65です。build.batを実行すると、nesファイルが作成されます。

推奨するカートリッジはMapper 0。PRG-ROM 32KB(FLASH)、CHR-RAM 8KB(SRAM)。
https://github.com/nicotakuya/diyfccartridge
